🔬 How CryptoShield AI Detects Wallet Fraud

🧠

1. AI-Powered Analysis (Groq Llama 3.3 70B)

After fetching raw transaction data, our system sends key metrics to Groq's ultra-fast Llama 3.3 70B model. The AI evaluates behavioral patterns, flags anomalies, and returns a comprehensive risk score (0–100) along with specific fraud classifications and actionable recommendations.

Groq Cloud · Llama 3.3 70B · 0.1 Temperature
📡

2. On-Chain Data Extraction

Ethereum: Etherscan API provides complete transaction history, balance, incoming/outgoing flows, and timestamps. Bitcoin: Blockstream API fetches UTXO data, transaction details, and address statistics. Both support caching for performance.

Etherscan API + Blockstream API + Redis-style Cache
⚠️

3. Multi-Layer Fraud Detection

Dusting Attacks: Identifies micro-transactions (<0.0001 ETH / 0.00001 BTC) that could be address poisoning. Money Laundering: Flags high outgoing tx volume to unique addresses. Mixing Services: Detects low balance with high transaction volume. Pattern Recognition: Analyzes hourly activity, counterparty diversity, and transaction size distribution.

Rule-Based Heuristics + Behavioral Analytics
CRITICAL (70-100) — Immediate threat, DO NOT interact
HIGH (50-69) — Extreme caution, verify independently
MEDIUM (25-49) — Suspicious patterns detected
LOW (10-24) — Minor anomalies, standard security
VERY LOW (0-9) — Normal activity patterns
